# Santiago Garrote's Dotfiles

This repository contains all the important dotfiles and scripts for setting up a Linux system with my stable configuration.

## Features
- Essential configuration files (dotfiles) for shell, Neovim, and other tools.
- A list of installed packages for reference.
- A setup script (`setup.sh`) that installs necessary packages and restores dotfiles.

## Usage
To set up a new system, simply run:

```bash
./setup.sh
